❌ المشكلة الأصلية:
--------------------
كان التطبيق يحاول الاتصال بـ localhost:
const SERVER_URL = 'http://127.0.0.1:5000';

هذا يسبب مشكلة لأن:
- على Railway: المستخدم يفتح مثلاً https://yourapp.railway.app
- لكن التطبيق يحاول الاتصال بـ http://127.0.0.1:5000
- النتيجة: Failed to fetch

✅ الحل المطبق:
--------------
تم تغيير SERVER_URL ليستخدم الـ URL الحالي تلقائياً:
const SERVER_URL = window.location.origin;

الآن التطبيق يشتغل على:
✓ localhost:5000 → يتصل بـ http://localhost:5000
✓ Railway → يتصل بـ https://yourapp.railway.app
✓ أي domain آخر → يتصل بنفس الـ domain
